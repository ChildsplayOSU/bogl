evalWithLimit :: Eval Val -> Eval Val Source #

Take an expressions, and before evaluating it checks & updates evaluation iterations If the count is less than the limit, continue evaluating, otherwise return an error instead of evluating further. Prevents infinite loops via recursion, while, or self referencing value equations, among other things
