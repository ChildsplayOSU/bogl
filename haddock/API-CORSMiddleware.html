allowCsrf :: Middleware Source #

x-csrf-token allowance. The following header will be set: Access-Control-Allow-Headers: x-csrf-token.

corsified :: Middleware Source #

appCorsResourcePolicy :: CorsResourcePolicy Source #

Cors resource policy to be used with corsified middleware.

This policy will set the following:

  • RequestHeaders: Content-Type
  • MethodsAllowed: OPTIONS, GET, PUT, POST
